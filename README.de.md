# Open Career Format (OCF)

[English](README.md) | [Español](README.es.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Português](README.pt.md) | [日本語](README.ja.md)

OCF ist ein offenes, von der Person kontrolliertes Dateiformat, das den beruflichen Werdegang bewahrt und für Lebensläufe, Anschreiben, Profile, Vorstellungsgespräche und Karrierewerkzeuge wiederverwendbar macht.

Diese Seite ist ein dauerhaftes deutschsprachiges Einstiegsportal, keine vollständige Übersetzung des Repositorys. Die maßgebliche technische Dokumentation wird auf Englisch gepflegt.

## OCF auf Deutsch verwenden

Beginnen Sie unter <https://opencareerformat.org/index.de.html>. Ein modernes LLM kann die maßgeblichen englischen Anweisungen befolgen und zugleich das Gespräch, die Fragen und die für die Person bestimmten Dokumente auf Deutsch verfassen. JSON-Schlüssel, schema-gesteuerte Werte und Validierungsregeln bleiben unverändert.

- [Deutscher Start-Prompt](prompts/application-bootstrap.de.md)
- [Maßgebliche englische Dokumentation](README.md)
- [Aktuelles Schema](schema.json)
- [Beispiel Maria Reyes](spec/examples/maria-reyes/README.md)

Es gibt kein Konto, keinen Dienst und nichts zu kaufen. Die OCF-Datei bleibt unter der Kontrolle der Person.
