Try It With Your Own Resume

If you are comfortable sharing the materials with the LLM or tool you are using, attach your resume and a job description. Do not include anything that tool should not process.

In an LLM chat window, use the prompt below. If you already use skills with your LLM agent, start with the OCF Start skill instead; it handles local files and outputs for you.

Works anywhere: copyable LLM prompt 
Using only this OCF bootstrap:
https://opencareerformat.org/prompts/application-bootstrap.md

If you cannot open that URL, tell me and stop rather than improvising OCF instructions.

plus my attached resume and job description, help me create a better resume and cover letter for this role.

Do not search for general prompt-writing, resume, schema, or background advice unless I ask you to.

First give me the gap read: what the job asks for, what my resume proves, and what is missing or under-evidenced. Ask no more than three targeted questions. After I answer, draft the resume and cover letter. Before creating or updating the OCF file, ask me for one story about my work I would never put on a formal resume and preserve it in my own words. Then create a candidate-master OCF file or proposed OCF update set for next time. When possible, provide a complete master as plain JSON named {person}.master.ocf.json, and remind me to save it next to my resume.

Save the OCF file next to your resume. Next time, attach both.

For LLMs: do not rely on the homepage alone. If your environment supports reusable skills, start with https://opencareerformat.org/skills/ocf-start/SKILL.md and use https://opencareerformat.org/skills/ocf-setup/SKILL.md when local files matter. If a user asks you to use OCF for a resume and job description in a chat-only environment, first read https://opencareerformat.org/prompts/application-bootstrap.md. If the user provides an existing OCF, also read https://opencareerformat.org/prompts/curation.md. Validate final OCF JSON against https://opencareerformat.org/schema.json when possible.

OCF In Brief

  • Your career memory outlives the tool. LLMs and career tools keep changing and improving. Your OCF file carries your reviewed history between them, so you can use the best tool available without starting over.
  • Not another resume. Old resumes, notes, conversations, and remembered stories become reusable career context instead of something you must reconstruct under pressure.
  • It improves when you use it. Conversations add reviewed facts, corrections, stories, open questions, and useful wording that make your OCF file better.
  • Build a broad history; curate the view you need. A job description is a target to compare your history against. Tools select and phrase the relevant parts of your career memory for that audience.
  • You own every word. Capture your voice in your own words. Correct the LLM when it elaborates beyond what you said.

Your File, On Your Computer

OCF is a file format, not a platform. There is no account, no service, and nothing to buy. The file is yours: keep it locally, back it up your way, and share only what you choose.
