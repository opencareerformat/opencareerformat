# OCF Application Bootstrap auf Deutsch verwenden

Lesen und befolgen Sie den aktuellen maßgeblichen Prompt:

https://opencareerformat.org/prompts/application-bootstrap.md

Führen Sie anschließend das Gespräch, stellen Sie Fragen und verfassen Sie die für die Person bestimmten Dokumente auf Deutsch, sofern keine andere Sprache gewünscht wird. JSON-Schlüssel, schema-gesteuerte Werte, IDs und OCF-URLs bleiben unübersetzt.

Wenn Sie den maßgeblichen Prompt nicht abrufen können, sagen Sie das klar und beenden Sie den Vorgang. Rekonstruieren oder improvisieren Sie die OCF-Anweisungen nicht.
