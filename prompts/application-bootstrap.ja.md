# OCF Application Bootstrap を日本語で使う

現在の正式なプロンプトを読み、その指示に従ってください。

https://opencareerformat.org/prompts/application-bootstrap.md

その後、別の言語が指定されない限り、会話、質問、個人向けの文書は日本語で作成してください。JSON キー、スキーマが定義する値、ID、OCF の URL は翻訳しないでください。

正式なプロンプトを取得できない場合は、そのことを明確に伝えて停止してください。OCF の指示を推測で再構成したり、独自に作ったりしないでください。
