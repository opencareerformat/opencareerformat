# Open Career Format (OCF)

[English](README.md) | [Español](README.es.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [Português](README.pt.md) | [日本語](README.ja.md)

OCF は、個人が管理するオープンなファイル形式です。職歴を保存し、履歴書、カバーレター、プロフィール、面接、キャリア支援ツールで再利用できます。

このページは安定した日本語の入り口であり、リポジトリ全体の翻訳ではありません。正式な技術文書は英語で管理されます。

## OCF を日本語で使う

<https://opencareerformat.org/index.ja.html> から始めてください。最新の LLM は、正式な英語の指示に従いながら、会話、質問、個人向けの文書を日本語で作成できます。JSON キー、スキーマが定義する値、検証ルールは翻訳しません。

- [日本語の開始プロンプト](prompts/application-bootstrap.ja.md)
- [正式な英語ドキュメント](README.md)
- [現行スキーマ](schema.json)
- [Maria Reyes の例](spec/examples/maria-reyes/README.md)

アカウントもサービスも購入も必要ありません。OCF ファイルは個人の管理下にあります。
